# Denny Ng ŌĆö Modern Portfolio

A bright, modern, and energetic personal portfolio built with **Next.js**, **Tailwind CSS**, **Framer Motion**, and **Swiper**.

Designed with a sunny color palette (warm oranges, soft peach, sunshine yellow and sky accents) to keep things positive and friendly while showcasing work and services.

## :toolbox: Getting Started

1. Make sure **Git** and **Node.js** are installed.
2. Clone this repository to your local machine.
3. From the project root, install dependencies:
   ```bash
   npm install --legacy-peer-deps
   # or
   yarn install --legacy-peer-deps
   ```
4. Start the dev server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## :art: Theme

The palette lives in [`tailwind.config.js`](./tailwind.config.js):

| Token       | Color      | Purpose                            |
| ----------- | ---------- | ---------------------------------- |
| `primary`   | `#0F172A`  | Text & dark UI elements            |
| `secondary` | `#FFF7ED`  | Warm cream background              |
| `accent`    | `#F97316`  | Main vibrant orange accent         |
| `sunrise`   | `#FB923C`  | Hover / secondary accent           |
| `coral`     | `#FB7185`  | Playful coral highlight            |
| `sunshine`  | `#FACC15`  | Bright yellow accent               |
| `sky`       | `#38BDF8`  | Soft blue for contrast             |
| `mint`      | `#34D399`  | Fresh green highlight              |

To customize the palette, edit the `colors` section in `tailwind.config.js`.
